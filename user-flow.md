# Big Sister AI — Kullanıcı Akışı

## Ana Akış

1. Kullanıcı uygulamayı açar → Karşılama ekranı gelir
2. "Başla" butonuna basar → Profil oluşturma ekranına geçer
3. Profil bilgilerini doldurur:
   - Kendi adını yazar
   - Yaşını girer
   - Ablanın yaş farkını seçer (1-10 yıl)
   - Ablanın uzmanlık alanını seçer
   - Ablanın adını belirler
4. "Ablama Kavuş" butonuna basar → Sohbet ekranına geçer
5. Sohbet ekranında mod seçer (Duygusal / Akademik / Hobi / Kariyer)
6. Ablayla sohbet eder
7. İstediği zaman modu değiştirebilir
8. İstediği zaman profili sıfırlayabilir

## Ekranlar

### 1. Karşılama Ekranı
- Uygulama adı ve logosu
- Kısa açıklama
- "Başla" butonu

### 2. Profil Oluşturma Ekranı
- Kullanıcı adı girişi
- Kullanıcı yaşı girişi  
- Yaş farkı seçimi (dropdown: 1-10)
- Uzmanlık alanı seçimi (Akademik / Spor / Müzik / Sanat / İş Hayatı)
- Abla adı girişi
- "Ablama Kavuş" butonu

### 3. Sohbet Ekranı
- Üstte abla profil bilgisi
- Mod seçim butonları (Duygusal / Akademik / Hobi / Kariyer)
- Mesaj geçmişi alanı
- Mesaj yazma kutusu
- Gönder butonu
